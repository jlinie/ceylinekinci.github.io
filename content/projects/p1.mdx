> A minimalist portfolio with playful interactions and a simple authoring flow.

## Overview
This site is my public notebook: projects, research notes, and a journal.  
Goals: **fast**, **low-maintenance**, **nice to use**, and **easy to publish**.

- **Framework:** Next.js (App Router) → **static export**
- **Authoring:** Markdown/MDX files under `/content`
- **Hosting:** GitHub Pages (via GitHub Actions)
- **Styling:** Tailwind CSS
- **Extras:** custom cursor, parallax wall on the home, menu/topbar state, color-palette cards
